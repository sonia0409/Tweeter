# Tweeter Project

Tweeter is a simple, single-page Twitter clone,however it demonstrates the both front-end and back-end skills:
- HTML, 
- CSS, 
- JS, 
- jQuery, and 
- AJAX front-end skills, 
- and their Node, Express back-end skills.
